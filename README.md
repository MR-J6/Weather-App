Daily Weather App
A sleek, high-performance weather application built with Flutter. This app provides real-time weather updates with a Dynamic UI that adapts to the local time and weather conditions of any city in the world.

Features:

Real-time Data: Fetches accurate weather information using WeatherAPI.

Dynamic Theming: The app's background, colors, and shadows automatically switch between Day and Night modes based on the city's local time.

Adaptive Icons: Weather icons change dynamically (Sun, Clouds, Storm, Rain, etc.) to match the current condition code.

Detailed Metrics: Displays Wind speed, Humidity, and Cloud percentage in a clean CardView layout.

State Management: Optimized performance using the Provider package for efficient data handling.

Refactored Codebase: Follows clean code principles with a modular folder structure (models, providers, widgets, utils).

Tech Stack:

Framework: Flutter

Language: Dart

State Management: Provider

API: WeatherAPI.com
